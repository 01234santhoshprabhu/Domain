# Domain Jan 2026 Dashboard

Static dashboard for viewing the overall domain list and candidate-wise domain course details.

## Files

- `index.html`
- `domain_list.csv`
- `student_score.csv`

Keep the two CSV filenames unchanged because the dashboard auto-loads them.

## GitHub Pages

1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Open repository Settings.
4. Go to Pages.
5. Set source to `Deploy from a branch`.
6. Choose `main` branch and `/root`.
7. Save.

GitHub will provide a Pages URL after the site is published.
